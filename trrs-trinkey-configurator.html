🎧 TRRS Trinkey Configurator

Configure input switches connected via TRRS jack with an audio splitter, then download the config file to your board.

This tool generates a config.py file for use with the matching code.py on an Adafruit TRRS Trinkey. The board.TIP pin is set as output-low to serve as ground. Connect your switches between TIP (sleeve of splitter) and any of the three input pins. To cycle through modes, press and hold the designated Mode Switch for the configured hold duration.
Uses a stereo/mic TRRS audio splitter to break out up to 3 independent switch inputs: SLEEVE, RING_1, and RING_2. See the Adafruit TRRS Trinkey pinout guide for wiring details.

Step 1 — Select active input pins

Choose which TRRS contacts you are using as switch inputs. Only selected pins will appear in the HID configuration below.

Step 2 — Choose the Mode Switch pin

One switch is designated to cycle through modes. Hold it down for the set duration to advance to the next mode. Its regular HID action still works on a normal press.

💡
1.0 s
How long the Mode Switch must be held before the mode changes — short enough to be reachable, long enough to avoid accidental triggers during normal use. 0.5–1.5 s is a good range for most users.

Step 3 — Configure HID actions per mode

Assign a keyboard, media, or mouse action to each active switch for each mode. The NeoPixel LED indicates the current mode.
