This guide walks you through setting up the Adafruit TRRS Trinkey as a configurable assistive switch interface using CircuitPython.

## What You'll Need

- Adafruit TRRS Trinkey ([product page](https://www.adafruit.com/product/5954))
- A USB-A port on your computer (the Trinkey plugs in directly)
- A TRRS audio splitter cable and up to 3 momentary switches with 3.5mm plugs
- The files from this project (see below)

** 
The files for steps 1 and 2 are already available in this repo for the most current version of circuitpython.**

## Step 1 ‚Äî Install CircuitPython
The TRRS Trinkey runs **CircuitPython**, a beginner-friendly version of Python for microcontrollers. You need to install it once before anything else.

### 1a. Download the CircuitPython firmware

Go to the official download page for the TRRS Trinkey:

**[https://circuitpython.org/board/adafruit_trrs_trinkey_m0/](https://circuitpython.org/board/adafruit_trrs_trinkey_m0/)**

Click **"Download .UF2 Now"** under the latest **stable** release. As of this writing that is **CircuitPython 10.1.4**. Save the `.uf2` file to your computer.

### 1b. Enter bootloader mode

1. Plug the Trinkey into a USB-A port on your computer.
2. **Double-tap** the small reset button on the board (tap it twice quickly, like a double-click).
3. A new drive called **`TRINKEYBOOT`** will appear on your computer ‚Äî this is the bootloader drive.

> **Tip:** If you miss the double-tap and the drive doesn't appear, try again. The timing takes a moment to get used to.

### 1c. Drag the firmware onto the board

Drag and drop the `.uf2` file you downloaded onto the **`TRINKEYBOOT`** drive. The board will automatically restart, and a new drive called **`CIRCUITPY`** will appear in its place.

You've now installed CircuitPython. ‚úÖ

## Step 2 ‚Äî Download the CircuitPython Library Bundle

Some libraries used by this project need to be copied manually. They come from the official Adafruit CircuitPython Library Bundle.

> **Important:** The bundle version must match your CircuitPython version. If you installed CircuitPython 10.x, download the **Bundle for Version 10.x**.

Go to: **[https://circuitpython.org/libraries](https://circuitpython.org/libraries)**

Download the bundle zip file (it will be named something like `adafruit-circuitpython-bundle-10.x-mpy-YYYYMMDD.zip`). Extract the zip ‚Äî inside you'll find a `lib` folder containing hundreds of `.mpy` library files.

1. Open the extracted bundle folder and navigate into the `lib` subfolder.
2. Find `neopixel.mpy` and `adafruit_pixelbuf.mpy`.
3. Copy **both files** to the `lib` folder on your `CIRCUITPY` drive.

> If the `lib` folder doesn't exist on `CIRCUITPY` yet, create it first.

After copying, your `CIRCUITPY/lib/` folder should contain at minimum:

## Step 4 ‚Äî Configure Your Switch Layout

Open the **TRRS Trinkey Configurator** (`trrs-trinkey-configurator.html`) in a web browser (Chrome or Edge recommended).

1. **Step 1 in the configurator** ‚Äî Select which TRRS pins you are using (Sleeve, Ring 1, Ring 2).
2. **Step 2 in the configurator** ‚Äî Choose which switch will act as the Mode Switch (used to cycle through modes by holding it down), and set how long it needs to be held.
3. **Step 3 in the configurator** ‚Äî Assign a keyboard shortcut, media key, or mouse button to each switch for each of the 4 available modes.
4. Click **"Download config.py"**.

## Step 6 ‚Äî Test Your Switches

Plug in your TRRS splitter and switches, then test each one. The LED colour tells you the current mode:

## Troubleshooting

**The `CIRCUITPY` drive doesn't appear after copying the firmware.**
Try unplugging and re-plugging the Trinkey. If it still shows `TRRSBOOT`, the firmware copy may not have completed ‚Äî try again.

**The NeoPixel doesn't light up / nothing happens.**
Check that `code.py` and `config.py` are in the root of `CIRCUITPY` (not inside a subfolder). Also confirm `neopixel.mpy` and `adafruit_pixelbuf.mpy` are in `CIRCUITPY/lib/`.

**I get an `ImportError` about `neopixel` or `adafruit_pixelbuf`.**
One or both library files are missing from `CIRCUITPY/lib/`. Go back to Step 3 and copy them from the bundle.

**Switches aren't responding.**
Double-check your wiring ‚Äî the TIP contact on the TRRS jack provides the ground. Each switch should bridge between TIP and one of the input contacts (Sleeve, Ring 1, or Ring 2). Make sure the pins you've wired match what's selected in the configurator.

**The board appears as `CIRCUITPY` but won't accept file changes (read-only).**
This is a known issue with some operating systems. Try ejecting and re-plugging the board, or use a different USB port. On Windows, avoid editing files directly ‚Äî copy the file to your desktop, edit it, then drag it back.

---

## Updating the Configuration

To change your switch assignments at any time, simply open the configurator again, make your changes, download a new `config.py`, and drag it onto the `CIRCUITPY` drive, replacing the old one. No re-flashing needed.
